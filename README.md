## 部署
1.第一种方式：下载项目文件夹，在本地分别运行vue和spring，将vue中的全局变量url改为自己后端的ip:端口号。

2.第二种方式： 下载部署文件中的jar包，在本地控制台使用java -jar xxx.jar运行，访问端口为8088（这样部署仅在本地有效）
